# Club Reference Images

This folder contains real online reference photographs, organized by the club
taxonomy requested for SwingSight:

```text
wood/
  driver/
  wood/
  hybrid/
iron/
  irons/
  wedges/
```

The images are stored unmodified. They are intentionally reference examples
only—one image per leaf is not enough to train or evaluate a CNN. Build the
training set separately with many varied views, clubs, backgrounds, and lighting
conditions, while preserving the license and attribution requirements.

For the `iron/` branch, capture a **close-up of the club head with the marking
in view**. The target is the stamped iron number (`1`–`9`) or wedge label/loft
(for example `PW`, `SW`, or `56`). The marking may be on the face, sole, or rear
cavity, depending on the club design; a grip-end butt-cap is not useful. The two
Callaway 8-iron references below show number-bearing views required for the
number CNN.
