# Notebooks

Exploratory notebooks and experiments.

- `01_golfdb_video_model_training.ipynb` — golf-swing video research and training.
- `02_authorized_driver_dataset_builder.ipynb` — validates, filters to the approved driver manufacturers, downloads from an approved direct-URL manifest, and splits driver-head images. It intentionally does not scrape or enumerate the USGA Equipment Database; use it only with USGA's written authorization or an official export.
